# Mattress PWA Client

React + Vite + Tailwind + PWA for customer and admin experiences.

## Scripts
- `npm run dev` — start dev server (default port 5173)
- `npm run build` — production build
- `npm run preview` — preview production build

## Env
Copy `.env.example` to `.env` and fill Firebase and API base.

## PWA
Uses `vite-plugin-pwa` with autoUpdate and manifest icons.
